#MongoDB

## Install mongodb on debian
https://docs.mongodb.com/manual/tutorial/install-mongodb-on-debian/

## Start MongoDB
```bash
sudo service mongod start
```
## Stop MongoDB
```bash
sudo service mongod stop
```
## Restart MongoDB
```bash
sudo service mongod restart
```
## Log
```
/var/log/mongodb/mongod.log
```
