Functional Programming

What is functional programming?

  • Functional programming is, at its most basic, a way of thinking that places an emphasis on using functions

  • It's not a matter of just applying functions to come up with a result, the goal is to compose everything.

  • Functional programming is about decomposing your programs into tiny reuseable functions that can be "composed" together to provide the specific functionality needed.

Thinking functionally

  • Extensible
  • Modular
  • Re-usable

Let's look at following example:

Take a list of intergers

  • 1. Add one to each value
  • 2. Times each value by 10
  • 3. Divide each value by 2

Use for loop:

        
          const myList = [1,2,3,4,5];

          for (var i = 0; i < myList.length; i++) {{ '{' }}
            myList[i] += 1;
            myList[i] *= 10;
            myList[i] /= 2;
         {{ '}' }}

wrap it into a function

        
          const myList = [1,2,3,4,5];

          const mapAddOneTimesTenDivideByTwo = list =>
            list.map(x=>
              ((x + 1) * 10 / 2)
            )
          
          mapAddOneTimesTenDivideByTwo(myList);

Let's refactor to functional:

        
          const myList = [1,2,3,4,5];

          // declare pure functions:

          const adOne = x => x + 1;
          const timesTen = x => x * 10;
          const divideByTwo = x => x / 2;


          // More Ramda fun

          const mapAddOneTimesTenDivideByTwo = 
            map(
              compose(
                divideByTwo,
                timesTen,
                adOne
              )
            )
