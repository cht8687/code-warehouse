From Merge Request to Code Review

Steps:

1. Rebase feature branch from latest delevelop: git rebase develop


2. Run the test: gulp test


3. Run the lint tool: gulp vet


4. Squash your commit message:


Commit message should be meaningful, considering following commit message:

  • [CC-9999]: fixes issues of the code reviews
  • [CC-9999]: fixes the code reviews
  • [CC-9999]: Add context menu for LHS panel menu

    • Format of commit message:

    [CC-XXXX]: Add context menu for LHS panel menu.

    5. Raise a merge request


    6. Notify reviewer


    7. Make changes and repeat step 1
